# JES
JustExplainStylesheets (jes, spoken like *jazz*, not *j.e.s.*) is a documentation generator for style sheets.

## Supported Formats
### Input (Style Sheets)
- CSS (default)
- SASS

### Output (Documentation)
- Markdown (default)
- HTML

## Installation (soon!)
```
pip install jes
```

## Usage (soon!)
Example:
```
jes style.css
```
